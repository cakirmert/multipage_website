Bereitstellung der Website

Diese Website wird als statische Website über GitHub Pages bereitgestellt. GitHub Pages kann beim Aufruf der Website technische Zugriffsdaten verarbeiten, insbesondere IP-Adresse, Datum und Uhrzeit des Abrufs, angeforderte Dateien, Referrer, Browsertyp, Betriebssystem und HTTP-Status. Diese Verarbeitung ist technisch erforderlich, um die Website auszuliefern und die Sicherheit des Dienstes zu gewährleisten.

Anbieter ist GitHub, Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA. Weitere Informationen enthält die GitHub Privacy Statement.

Client-seitige Berechnungen

Die Eingaben in die Rechner werden im Browser verarbeitet. Der Betreiber dieser Website speichert diese Eingaben nicht und betreibt keine eigene Datenbank, kein Nutzerkonto-System, keine Formulare und kein Tracking.

Die Anwendung verwendet keine Analyse-Cookies und keine Werbe- oder Marketingdienste. Browserseitige Abrufe von lokalen JSON-, Markdown-, WebAssembly- und Bilddateien dienen ausschließlich der Funktion der Rechner und Dokumentation.

Externe technische Ressourcen

Einige fachliche Textbereiche laden zur Darstellung von Markdown und Formeln technische Bibliotheken von jsDelivr/CDN-Anbietern nach (Marked und KaTeX). Beim Abruf dieser Dateien kann der jeweilige CDN-Anbieter technische Zugriffsdaten erhalten. Die rechtlichen Seiten selbst sind statisches HTML und laden diese externen Bibliotheken nicht.
