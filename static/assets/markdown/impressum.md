## Hosting

Diese Website wird als rein statische Anwendung ausgeliefert. Sämtliche
Berechnungen (PHREEQC‑Speziation, Karbonatsystem etc.) laufen über
WebAssembly direkt im Browser der Nutzerinnen und Nutzer. Es werden
keine personenbezogenen Daten an den Server übertragen oder dort
verarbeitet.

Hosting‑Anbieter: **Cloudflare Pages** (alternativ: DigitalOcean App
Platform). Beim Aufruf der Seite werden vom Hosting‑Anbieter technisch
erforderliche Verbindungsdaten (insbesondere IP‑Adresse, Datum und
Uhrzeit, Browser‑Kennung) für die Dauer der Auslieferung verarbeitet;
eine darüber hinausgehende Auswertung oder Speicherung durch den
Betreiber dieser Website findet nicht statt.
