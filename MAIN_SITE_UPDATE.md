The live `enhanced-weathering.de` source is `cakirmert/lukas`, not
`cakirmert/cakirmert.github.io`. The tools link was added there in commit
`e0cc329` (`Add Weathering Tools link to main site`).

```text

```

```text
About | Services | Research Tools | Credentials | Contact
```

## Homepage Section

Place this after the hero or before services:

```text
Research Tools

Weathering Tools is a geochemistry toolbox for enhanced-weathering research. It
includes carbonate-system calculators, PHREEQC speciation, charge-balance
checks, mineral dissolution helpers, Bjerrum plots, and XRF oxide conversion
tools.

Open Weathering Tools -> https://tools.enhanced-weathering.de/
```

## Why This Should Live on the Main Site

The tools make the consulting/research site more useful for researchers without
turning the main homepage into a heavy application. Keeping the tools on
`tools.enhanced-weathering.de` gives Lukas one clear public identity while
allowing the calculator suite to deploy independently through GitHub Pages.
